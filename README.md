# Pussy University — Offer Portal (Final Project)

A classic academic-style website that presents the latest application update and an offer of admission letter.

## Files
- `index.html` — landing page + offer letter
- `style.css` — traditional, paper-like academic styling
- `script.js` — basic interaction (view application, confirm enrollment, print)
- `assets/pu-logo.jpeg` — crest/logo (provided)

## Run locally
Just open `index.html` in a browser.

## Deploy on GitHub Pages
1. Create a new GitHub repo.
2. Upload all files in this folder to the repo root.
3. In GitHub: **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` / `(root)`
4. Save. Your site will publish.

## Interaction
- **View the Application** scrolls to the offer letter section.
- **Confirm Enrollment** becomes available after acknowledgment; clicking applies the official stamp and updates status.
- **Print** prints a paper-style copy.
